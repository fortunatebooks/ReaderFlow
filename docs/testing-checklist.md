# ReaderFlow Testing Checklist

## Automated Checks

- Generate project with `make generate`.
- Build with `make build`.
- Run unit/UI tests with `make test`.

## Device Smoke Test

- Import a DRM-free EPUB.
- Open the book.
- Start and pause autoscroll.
- Change speed.
- Select text and save an excerpt.
- Reopen the book and verify auto-resume.
- Export excerpts.
