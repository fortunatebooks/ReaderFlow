# Bundled Reader Fonts

ReaderFlow bundles these fonts for offline reader typography:

- Atkinson Hyperlegible from `googlefonts/atkinson-hyperlegible`, licensed under SIL OFL 1.1.
- Literata from `googlefonts/literata`, licensed under SIL OFL 1.1.
- Source Serif 4 from `adobe-fonts/source-serif`, licensed under SIL OFL 1.1.

License files are stored beside each font family.
